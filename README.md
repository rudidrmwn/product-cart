# SHOPING CART USING LARAVEL 10 AND VUE
### Installing 
```
composer install
npm install
```

### Executing program

### Setup Environment Variable
```
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=databasename
DB_USERNAME=root
DB_PASSWORD=
```

### Migration Table
```
php artisan migrate
```
### Migration Data Seed
```
php artisan db:seed
```
### Running 
```
php artisan serve   
npm run dev
```
